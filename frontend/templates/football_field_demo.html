Usage Examples

Basic Implementation

// Create a standard field
const field = new FootballField('field-container', {
    width: 800,
    height: 520,
    interactive: true,
    showPlayerPositions: true,
    formation: '4-3-3'
});

// Handle player clicks
field.options.onFieldClick = (x, y) => {
    console.log('Field clicked at:', x, y);
};
